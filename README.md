## Overview

CyberPath is a modern, interactive web application designed to help students master cybersecurity concepts. Moving beyond traditional static textbooks, CyberPath offers rich visualisations, real-world analogies, and hands-on practice modules in a stunning, highly responsive "glassmorphism" user interface.

## 📚 Curriculum Units

1. **Introduction to Cyber Security**  
   Explore the foundations of cybersecurity — the CIA triad, attack types, security mechanisms, and the basics of symmetric encryption.
2. **Public-Key Cryptography & Message Authentication**  
   Dive into hash functions, public-key cryptography, RSA, digital signatures, and the foundations of key management and PKI.
3. **Authentication Applications**  
   Master real-world authentication systems — Kerberos protocol, X.509 certificates, PKI infrastructure, and email security with PGP and S/MIME.
4. **Network Management Security**  
   Understand network management protocols (SNMP), intrusion detection systems, password security, and practical defense mechanisms against network threats.
5. **Malicious Software**  
   Analyze the full malware landscape — viruses, worms, trojans, ransomware, DDoS attacks, detection techniques, and firewall architectures for defense.

## ✨ Key Features

- **Interactive Visualisations**: Watch data flow through Kerberos, simulate RSA encryption step-by-step, or see how an IDS flags malicious traffic.
- **Hands-On Widgets**: Try your hand at Caesar ciphers, hash generation, and firewall rule building right from your browser.
- **Progress Tracking**: A gamified lesson trail keeps track of your completion status and encourages you to finish the course.
- **Premium Modern UI**: Built with a sleek dark mode, frosted glass elements, and subtle micro-animations for an engaging learning environment.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

To run CyberPath locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Aliasgar-Jiwani/cyberpath.git
   cd cyberpath
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to `http://localhost:5173` (or the port specified in your terminal).
